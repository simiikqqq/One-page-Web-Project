/assets                        – Statické soubory pro frontend (styly, skripty, obrázky)
    /css                       – Stylovací soubory (globální styly, layout, UI)
    /img                       – Obrázky a ikony používané v aplikaci
    /js                        – JavaScript logika na straně klienta (DOM, AJAX, práce s daty)

/data                          – Datová složka sloužící jako jednoduchá databáze JSON souborů
    archive.json               – Archivované poznámky, které nejsou aktivní
    categories.json            – Kategorie/tagy používané pro třídění poznámek
    notes.json                 – Hlavní soubor obsahující list aktuálních poznámek

/docs                          – Dokumentace projektu pro vývoj, design i API
    api_notes.md               – Dokumentace API nebo datové komunikace
    data_structure.md          – Popis struktury JSON dat a datových modelů
    design_notes.md            – Návrh UI/UX, barvy, layout, wireframy
    one-page_web.md            – Stručný popis projektu

/public                        – Soubory veřejně přístupné pro prohlížeč
    index.html                 – Hlavní HTML stránka aplikace

README.md                      – Úvodní dokumentace projektu, návod ke spuštění, popis funkcí
STRUCTURE.md                  – Tento dokument popisující strukturu celého projektu
