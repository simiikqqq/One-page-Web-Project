## **Název projektu**

**Simple Notes App**

## **Žák**
Šimon Novák

## **Stručný popis tématu (3–5 vět)**

Simple Notes App je jednoduchá webová stránka, kde si můžeš psát a ukládat své poznámky, zápisy nebo nápady. Umožní ti vytvářet nové záznamy, upravovat je a přehledně si je třídit do různých skupin. Poznámky se ukládají na server, takže o ně nepřijdeš, ani když stránku zavřeš. Aplikace je přehledná, dobře se ovládá na počítači i na mobilu a mezi jednotlivými částmi se dá jednoduše přepínat. Je to praktický nástroj pro každodenní používání.

## **Cílová skupina**

Cílovou skupinou aplikace jsou lidé, kteří potřebují jednoduché a přehledné místo pro ukládání svých poznámek, ať už jde o studenty, pracující nebo běžné uživatele hledající praktický a jednoduchý nástroj pro každodenní zapisování.

## **Návrh obsahových sekcí**

1. **O aplikaci** – základní představení jednoduché webové aplikace pro ukládání poznámek a zápisů.
2. **Seznam poznámek** – přehled všech uložených poznámek s možností je rychle otevřít nebo vyhledat.
3. **Editor poznámek** – stránka pro psaní, úpravu a ukládání textových zápisů.
4. **Kategorie (tagy)** – rozdělení poznámek do skupin pro lepší přehlednost.
5. **Archiv** – místo pro staré nebo odložené poznámky, které už nejsou aktivní.



## **Návrh designu / wireframe**

- Jednoduché světlé barevné ladění (šedá, modrá, jemné pastelové odstíny).
- Přehledné blokové uspořádání sekcí s viditelnou horní navigací.
- Seznam poznámek zobrazený v čistém listu nebo kartách s krátkým náhledem.
- Editor poznámek s velkou textovou plochou a hlavními tlačítky (Uložit / Upravit / Smazat).
- Responzivní rozvržení – na menších obrazovkách se obsah skládá pod sebe, navigace se zjednoduší.

## **Použité technologie**

- **Frontend:** HTML, CSS nebo Tailwind CSS, JavaScript
- **Data:** JSON soubory pro poznámky, kategorie a archiv
- **AJAX:** načítání a ukládání poznámek bez obnovování stránky
- **Backend:** jednoduchý PHP nebo Node.js endpoint pro práci s daty
- **Verzování:** Git + GitHub (repozitář dostupný vyučujícímu)

## **Očekávaný přínos / co se naučím**
